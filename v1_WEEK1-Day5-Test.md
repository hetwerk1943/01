## 🔥 DZIEŃ 5 — Test w przeglądarce

1. Otwórz Chrome i wejdź na:
   ```
   chrome://extensions
   ```
2. Włącz **Tryb deweloperski** (prawy górny róg).
3. Kliknij **Załaduj rozpakowane**.
4. Wskaż folder:
   ```
   extension/
   ```

---

**Jeśli:**
- Popup się otwiera
- Dashboard się ładuje
- Brak błędów w konsoli
