## 📦 Instalacja (wersja developerska)

```bash
git clone https://github.com/YOUR_USERNAME/LifeHub.git
cd LifeHub
npm install
npm start
```

Wersja MVP bazuje na Chromium (rozszerzenie Manifest V3).
