# Security Policy

## Supported Versions

The following versions of Ultra Security Monitor receive active security updates:

| Version (PowerShell requirement) | Supported          |
| -------------------------------- | ------------------ |
| PowerShell 5.1+                  | :white_check_mark: |
| PowerShell < 5.1                 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please **do not** open a public GitHub issue.

Instead, report it privately:

1. Open a [GitHub Security Advisory](https://github.com/hetwerk1943/-Ultra-Security-Monitor-Wersja-Totalna-Full-Suite-/security/advisories/new) in this repository, **or**
2. Send a description of the vulnerability to the maintainer via GitHub ([@hetwerk1943](https://github.com/hetwerk1943)).

Please include:
- A clear description of the vulnerability and its potential impact.
- Steps to reproduce the issue.
- Any suggested mitigations or fixes.

You can expect an acknowledgement within **48 hours** and a resolution or status update within **7 days**. Once a fix is confirmed, a patched release will be published and credit given to the reporter (unless anonymity is requested).
